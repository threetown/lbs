## 数据总览接口

### 一、流量分析

* 请求地址：`/api/...`

* 请求方式：`POST`

* 请求参数：

  ```json
  {
      "type": 1,       // 类型: 1，访问次数；2,访问人数
      "starttime":"",   // 开始时间
      "endtime":""     // 结束时间
  }
  ```

* 返回结果：

  ```json
  {
      "data": [120, 132, 101, 134, 90, 230, 210]
  }
  ```

### 三、用户分析

- 请求地址：`/api/...`
- 请求方式：`POST`

- 请求参数： 

  ```json
  {
      "starttime":"",   // 开始时间

  }
  ```

- 返回结果：


### 四、访问统计
